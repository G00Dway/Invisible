# Invisible
Invisible, wifi (DEAuth) tool made for pentesers and ethical hackers
# Requirements
- [X] Working wifi interface
- [X] Root permissions
# Installation:
`git clone https://github.com/G00Dway/Invisible`
>
`cd Invisible`
>
`pip install -r requirements.txt`
>
`python wifi.py`
